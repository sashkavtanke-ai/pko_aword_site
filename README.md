# Сайт для коллекторского агентства "Аворд
Этот проект представляет собой веб-сайт, разработанный для коллекторского агентства "Аворд". Сайт предоставляет информацию об услугах агентства, юридическую информацию, а также позволяет пользователям проверить свою задолженность и связаться с представителями компании.

## Технологический стек

*   **React** - Основная библиотека для построения пользовательского интерфейса.
*   **Vite** - Инструмент для сборки и разработки, обеспечивающий быструю горячую перезагрузку (HMR).
*   **React Router** - Для навигации по страницам сайта.
*   **React Bootstrap** - Компоненты пользовательского интерфейса, стилизованные под Bootstrap.
*   **ESLint** - Для статического анализа кода и поддержания его качества.

## Структура проекта

*   `public/` - Статические файлы, такие как изображения и документы.
*   `src/` - Исходный код приложения.
    *   `components/` - Переиспользуемые React-компоненты (Header, Footer, формы и т.д.).
    *   `pages/` - Компоненты, представляющие отдельные страницы сайта.
    *   `styles/` - Файлы стилей CSS.
    *   `App.jsx` - Корневой компонент приложения, где настраивается маршрутизация.
    *   `main.jsx` - Точка входа в приложение.

## Установка и запуск

1.  **Установите зависимости:**
    ```bash
    npm install
    ```

2.  **Запустите в режиме разработки:**
    ```bash
    npm run dev
    ```
    Сайт будет доступен по адресу `http://localhost:5173`.

3.  **Сборка для продакшена:**
    ```bash
    npm run build
    ```
    Собранные файлы будут находиться в директории `dist/`.

4.  **Запуск превью продакшн-сборки:**
    ```bash
    npm run preview
    ```

## Линтинг

Для проверки качества кода выполните команду:

```bash
npm run lint
```

## Docker

Проект также может быть запущен в Docker-контейнере.

1.  **Сборка образа:**
    ```bash
    docker build -t pko-avord .
    ```

2.  **Запуск контейнера:**
    ```bash
    docker run -p 80:80 pko-avord
    ```
    Сайт будет доступен по адресу `http://localhost`.
